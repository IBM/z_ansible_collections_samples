register_with_dbrc
=========

Register the IMS catalog to the IMS Database Recovery Control facility (DBRC) by using the `ims_dbrc` module in the `ibm_zos_ims` Ansible collection.

Requirements
------------

[IBM&reg; z/OS&reg; core collection 1.4.0 or later](https://galaxy.ansible.com/ibm/ibm_zos_core)

[IBM&reg; z/OS&reg; IMS collection 1.2.0](https://galaxy.ansible.com/ibm/ibm_zos_ims)

Role Variables
--------------

None

Dependencies
------------

None
